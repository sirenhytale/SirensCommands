<span style="color: #169179;">===--===--===--===--===--===--===--===--===--===--===--===--===--===--===--===--===</span>

<span style="color: #bfedd2;">Siren's Commands is a plugin which is designed to add numerous useful commands to your server to make your life more easy.&nbsp; With sights to include other content than commands to make improve your experience.</span>

<span style="color: #bfedd2;"><strong>This plugin has </strong><span style="text-decoration: underline;"><strong>no direct relation</strong></span><strong> with my other plugin called </strong><strong><a style="color: #bfedd2;" href="https://www.curseforge.com/hytale/mods/mermaids" target="_blank" rel="nofollow noopener">Mermaids</a>.</strong></span>

<span style="color: #bfedd2;"><strong>This plugin is in active development and many other commands will be added in the future.</strong></span>

<span style="color: #169179;">===--===--===--===--===--===--===--===--===--===--===--===--===--===--===--===--===</span>

## <span style="color: #2dc26b;"><strong>Commands</strong>:</span>

*   <span style="color: #bfedd2;">/sirenscommands, /scmds : to view all commands on the server</span>
*   <span style="color: #bfedd2;">/list, /players : See who is connected to the server</span>
*   <span style="color: #bfedd2;">/gmc, /gma : Sets the player's gamemode</span>
*   <span style="color: #bfedd2;">/top : Sends player to the surface</span>
*   <span style="color: #bfedd2;">/sucide, /die : Kills the player</span>
*   <span style="color: #bfedd2;">/back : Teleports the player to their previous location</span>
*   <span style="color: #bfedd2;">/day, /night : More accurate time to set day or night</span>
*   <span style="color: #bfedd2;">and more coming soon!</span>

<span style="color: #169179;">===--===--===--===--===--===--===--===--===--===--===--===--===--===--===--===--===</span>
